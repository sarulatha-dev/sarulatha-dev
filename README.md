### 🌟 WorkEtiq — AI-Powered Workplace Simulator

</div>

> *Bridging the gap between academic knowledge and real-world corporate skills through gamified AI coaching.*

**📌 Overview**

WorkEtiq is an interactive, gamified workplace simulation platform that uses real-time AI coaching to help users master professional communication, negotiation, and corporate decision-making — in a safe, engaging environment.

**⚙️ Tech Stack**

![Next.js](https://img.shields.io/badge/Next.js-000000?style=flat-square&logo=next.js&logoColor=white)

![Tailwind CSS](https://img.shields.io/badge/Tailwind_CSS-38B2AC?style=flat-square&logo=tailwind-css&logoColor=white)
![Claude AI](https://img.shields.io/badge/Claude_AI-CC785C?style=flat-square&logo=anthropic&logoColor=white)

**✨ Key Features**

| Feature | Description |
|---|---|
| 🎭 Interactive Drama Episodes | Decision-making scenarios with branching storylines |
| 🤖 AI Salary Negotiation Simulator | Real-time AI feedback on negotiation strategies |
| 🧠 5 Whys Analysis Tool | Guided root cause problem-solving framework |
| 📊 Gamified Dashboard | XP system, streaks, badges & progress tracking |
| 📅 Meeting Manager | AI-generated summaries and action items |

**👩‍💻 My Contribution**

> Sole Developer — designed UI/UX, architected the backend with Prisma + SQLite, integrated Claude AI APIs for dynamic feedback, and built all application logic and state management end-to-end.
