<div align="center">

```
╔══════════════════════════════════════════════════════════════╗

║    Bridging academic knowledge & real-world corporate skills ║
╚══════════════════════════════════════════════════════════════╝
```

</div>

> **Role:** Sole Developer (End-to-End) &nbsp;|&nbsp; **Stack:** Next.js · TypeScript · Prisma · SQLite · Claude AI · Zustand · Framer Motion

**The problem it solves:** Fresh graduates struggle with real workplace scenarios — salary negotiation, corporate communication, decision-making under pressure. WorkEtiq is the simulator that bridges that gap.

<details>
<summary><b>🔍 Feature Breakdown (click to expand)</b></summary>

<br/>

| # | Feature | What it does |
|---|---------|-------------|
| 🎭 | **Interactive Drama Episodes** | Branching decision-making storylines with consequence-driven outcomes |
| 🤖 | **AI Salary Negotiation Simulator** | Real-time Claude AI feedback on your negotiation strategy and language |
| 🧠 | **5 Whys Analysis Tool** | Guided root cause analysis with AI-coached problem solving |
| 📊 | **Gamified Dashboard** | XP system, daily streaks, achievement badges, and progress heatmaps |
| 📅 | **Meeting Manager** | AI-generated meeting summaries, action items, and follow-up reminders |

</details>
